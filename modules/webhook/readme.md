Webhooks
====

Получить уведомление о входящем звонке и отобразить уведомление менеджеру Стата. 
http://confluence.welltime.ru/pages/viewpage.action?pageId=8913103

Установка
---------

Для работы необходим установленный и запущенный socket-сервер.

Настройка
---------

Скопировать modules/webhook/config/params.php в modules/webhook/config/params.local.php и указать secretKey.
