GLPI
====

CRM-система. Лицензия GNU GPL. Подробнее см. http://glpi-project.org/spip.php?lang=en

Установка
---------

1. Скачать с http://glpi-project.org/spip.php?article41&lang=en
2. Разархивировать, настроить любой домен
3. В браузере открыть домен, пройти визард


Настройка API
-------------

* "Настройки" / "Общие" / "API "
    * "Включен Rest API" - Да
    * "Включить вход в систему с учетными данными" - Нет
    * "Включить вход с внешним идентификатором" - Да
    * "Добавить клиент API" 
        * "Наименование" - любое. Например, "stat"
        * "Токен приложения (app_token)" / "Обновить"
        * "Добавить"
        * Скопировать "Токен приложения (app_token)", указать его в параметре appToken конфига в stat (см. ниже) 

* "Администрирование" / "Пользователи" / "Добавить пользователя ..."
    * Выдать необходимые права
    * "Настройки"
        * "Ключ удалённого доступа" / "Обновить"
        * "Сохранить"
        * Скопировать "Ключ удалённого доступа", указать его в параметре userToken конфига в stat (см. ниже)

* Донастроить nginx по инструкции http://ДОМЕН/apirest.php#servers-configuration

* В stat скопировать modules/glpi/config/params.php в modules/glpi/config/params.local.php и указать
    * Адрес GLPI-сайт вместе со скриптом API. Например, 'https://glpi.mcn.ru/apirest.php'
    * appToken (см. выше "Токен приложения (app_token)")
    * userToken (см. выше "Ключ удалённого доступа")
